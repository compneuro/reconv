This is the README for ReConv algorithm.  

ReConv algorithm reconstructs evoked LFP in cerebellar granular layer.  


How-to:  
---------  
 
Change current working directory to 'data\Invitro' and run "weighsum.m" to generate in vitro evoked LFP as reported in Fig. 5C.  

Change current working directory to 'data\Invivo' and run "weighavg.m" to generate in vitro evoked LFP as reported in Fig. 6B. 

In case of any errors while running the scripts, please clear all variables in memory and rerun the scripts. 

This set of codes have been tested in Matlab 2009b, 2010b and in GNU octave 3.4.
